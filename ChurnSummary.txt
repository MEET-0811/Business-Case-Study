# 📌 Churn Summary

The Gym Membership Churn Analysis project focuses on identifying customer churn patterns 
and analyzing customer behavior using SQL and Power BI.

The project analyzes customer subscription details, login activity, session participation, 
and feedback scores to understand why customers leave gym memberships.

Using SQL queries, active customers, churned customers, churn rates, 
and inactive users were identified. 
The data was then visualized in Power BI using KPI cards, donut charts, bar charts, line charts, and scatter plots.

Key findings from the project showed that:

* Monthly subscription users had the highest churn rate
* Customers with low feedback scores were more likely to churn
* Customers with fewer sessions showed lower engagement
* Yearly subscribers were more loyal and active

This dashboard helps businesses improve customer retention, monitor customer engagement, 
and make better business decisions through data-driven insights.
